# PNG_MNIST_Prepper
This code is helpful to prepare PNG images to feed them into a neural network, without using any external libraries aside from Numpy.
This is especially handy if you wish to implement it on embedded devices running CircuitPython, MicroPython or other embedded versions of Python.
Numpy can easily be swapped out for derivatives specifically made for microcontrollers and SBCs.
The intention for this program is to use as little resources as possible, as embedded devices don't tend to be very powerful.
